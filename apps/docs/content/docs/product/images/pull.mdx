# Pull Images

Click **Pull Image** to download a new image from a registry.

## Required fields

- **Registry**: Docker Hub, GHCR, GitLab, ACR, GAR, or ECR Public.
- **Name**: The image name (e.g. `library/nginx` or `my-org/app`).
- **Tag**: The tag to pull (e.g. `latest`, `1.2.0`).

## Example

```
docker.io/library/nginx:latest
```

After the pull completes, the image appears in the table.
