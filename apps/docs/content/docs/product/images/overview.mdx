# Images Overview

The Images page lists all images on the host and shows detailed information
about each one.

## Images table

The table includes:

- **Image name** and **tag**
- **Size**
- **Architecture / OS**
- **Layers**

Select a row to view its details on the right panel.

## Image details

The details panel shows:

- Full image information
- Image size and layers
- Containers currently using the image

If containers are listed, they are still referencing that image.
